Профессиональное десктопное приложение для безопасного шифрования и расшифровки файлов с использованием гибридной криптографии (RSA + AES). Создано на Python и PyQt6.

## Возможности

### Функции безопасности
- **Гибридное шифрование**: RSA-4096 + AES-256-GCM
- **Безопасное управление ключами**: Приватные ключи защищены паролем
- **Аутентифицированное шифрование**: Режим GCM для обеспечения целостности данных
- **Формирование ключа**: PBKDF2 с 480 000 итераций

### Пользовательский интерфейс
- **Современные темы**: Профессиональный внешний вид. Поддержка светлой и темной тем
- **Мультиязычность**: Английский и русский интерфейс
- **Вкладочная структура**: Интуитивно понятная организация
- **Отслеживание прогресса**: Индикаторы выполнения (прогресс-бары) в реальном времени
- **История операций**: Полный журнал активности

## Как использовать

### 1. Генерация ключей
- Перейдите на вкладку "Управление ключами" (Key Management)
- Введите имя пары ключей
- Установите надежный пароль
- Выберите директорию для сохранения ключей
- Нажмите "Сгенерировать пару ключей" (Generate Key Pair)

### 2. Шифрование файлов
- Перейдите на вкладку "Шифрование" (Encryption)
- Выберите файл для шифрования
- Выберите публичный ключ из выпадающего списка
- Укажите путь для сохранения выходного файла
- Нажмите "Зашифровать" (Encrypt)

### 3. Расшифровка файлов
- Перейдите на вкладку "Расшифровка" (Decryption)
- Выберите зашифрованный файл
- Выберите приватный ключ из выпадающего списка
- Введите пароль
- Укажите путь для сохранения выходного файла
- Нажмите "Расшифровать" (Decrypt)

## Технические детали

### Криптография
- **Асимметричное шифрование**: RSA-4096 с OAEP-паддингом
- **Симметричное шифрование**: AES-256-GCM
- **Формирование ключа**: PBKDF2-HMAC-SHA256
- **Генерация случайных чисел**: Криптографически стойкие случайные байты

### Формат файла
```
[16 байт - IV (вектор инициализации)][16 байт - тег GCM]
[4 байта - длина зашифрованного ключа][N байт - зашифрованный сессионный ключ][остальное - зашифрованные данные]
```

### Хранение ключей
- **Публичные ключи**: Стандартный формат PEM
- **Приватные ключи**: Зашифрованы с помощью AES-GCM

## Устранение неполадок

### Частые проблемы

**Пустые выпадающие списки**
- Убедитесь, что выбрана директория с ключами
- Сначала сгенерируйте ключи
- Используйте кнопку "Обзор" (Browse), чтобы заново выбрать директорию с ключами

**Ошибки шифрования/расшифровки**
- Проверьте права доступа к файлам
- Убедитесь в действительности ключей
- Убедитесь в правильности пароля от приватного ключа

**Приложение не запускается**
- Проверьте версию Python (требуется 3.8+)
- Убедитесь, что установлены все зависимости
- Запустите приложение из терминала, чтобы увидеть сообщения об ошибках
